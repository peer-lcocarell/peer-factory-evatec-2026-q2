# R10.2 Update for CR 286067 - View Magnet System Documents: Verify uploaded magnet system documents are accessible (Happy Path)

## Requirement
CR: 286067
Ticket: 35008
Requirement ID: 290768
Requirement: View magnet system documents (View Magnet System)

## Preconditions
- User is signed in with permission to view Magnet Systems.
- A Magnet System exists with at least one attached document.
- View Magnet System page is accessible.

| Step | Action | Expected Result |
|------|----------|----------------|
| 1 | Open the View Magnet System page for a record with documents. | View screen loads with documents section visible. |
| 2 | Review the documents list. | Existing documents are listed with expected metadata. |
| 3 | Open/download one of the documents. | Document opens or downloads successfully. |
| 4 | Verify document content matches the expected file. | Document content is accurate and complete. |

## Expected Outcome
Uploaded Magnet System documents are viewable/downloadable from the View screen.
