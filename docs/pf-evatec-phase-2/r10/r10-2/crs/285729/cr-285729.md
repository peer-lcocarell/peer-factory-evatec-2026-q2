### CR 285729 – Search HW Components

**High-Level Summary**

This CR adds a new **Hardware Search** feature that helps users quickly locate where hardware components are installed within the factory configuration.

Users can search by:

* Part Number
* Description
* Serial Number
* Installation date range
* Docked or undocked module scope

The search results display the associated **Tool** and **Module** where the hardware is installed and provide navigation links to open those locations directly.

### Business Value

* Simplifies locating hardware across tools and modules.
* Reduces time spent manually reviewing hardware configurations and logbook records.
* Supports both current and historical hardware installation tracking.
* Improves troubleshooting and equipment management.

### Key Areas Impacted

* New Hardware Search dialog
* Tools page
* Modules page
* Tool Overview
* Hardware installation history and logbook data

### QA Focus

* Search accuracy across all supported filters.
* Historical date-range searches.
* Current versus historical installation results.
* Tool and Module navigation links.
* Correct handling of module and submodule locations.

### Risk Level

**Medium**

The feature primarily introduces new search and reporting functionality but relies on historical hardware installation records and module hierarchy data. Incorrect data relationships could result in inaccurate search results or location reporting.
