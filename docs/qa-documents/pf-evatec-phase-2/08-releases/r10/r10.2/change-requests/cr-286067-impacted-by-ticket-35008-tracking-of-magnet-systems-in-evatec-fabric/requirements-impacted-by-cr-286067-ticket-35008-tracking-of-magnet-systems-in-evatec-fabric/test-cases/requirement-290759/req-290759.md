## 13.4.3 Requirement: View Magnet System Type

If a user has permissions to view Magnet System Types but not edit, they can drill into the Magnet System Type details.

If the Magnet System Type is a prototype, the Variant Code is displayed using the format `V##`.

A user can open or download any documents uploaded for a Magnet System Type.

Additionally:

- A user can edit the Magnet System Type if they have the `MagnetSystemTypes_Edit` security right.
- A user can define the Magnet System Type as standard if it is a prototype and they have the `MagnetSystems_Modify` security right.
- Users must have the `MagnetSystemTypes_View` security right to access this functionality.

If a user has permissions to view Magnet System Types but not edit, they can drill into the Magnet System Type details.

If the Magnet System Type is a prototype, the Variant Code is displayed using the format `V##`.

A user can open or download any documents uploaded for a Magnet System Type.

Additionally:

- A user can edit the Magnet System Type if they have the `MagnetSystemTypes_Edit` security right.
- A user can define the Magnet System Type as standard if it is a prototype and they have the `MagnetSystems_Modify` security right.
- Users must have the `MagnetSystemTypes_View` security right to access this functionality.

## QA Audit Notes (2026-07-13)

### Coverage Status

Partial

### Active Mapped Test Cases

- 001-view-magnet-system-type-details-happy-path.md

### Confirmed Covered

- User with `MagnetSystemTypes_View` permission can access the Magnet System Types list.
- User can select an existing Magnet System Type.
- User can open the View page for a Magnet System Type.
- Magnet System Type details are displayed.
- Stored field values are displayed correctly.

### Coverage Gaps

#### Security Validation

- Verify a user with `MagnetSystemTypes_View` and without `MagnetSystemTypes_Edit` can view details but cannot edit the record.
- Verify edit functionality is available to users with `MagnetSystemTypes_Edit`.
- Verify edit functionality is unavailable to users without `MagnetSystemTypes_Edit`.
- Verify Define as Standard functionality is available to users with `MagnetSystems_Modify`.
- Verify Define as Standard functionality is unavailable to users without `MagnetSystems_Modify`.
- Verify users without `MagnetSystemTypes_View` cannot access Magnet System Type details.

#### Prototype Validation

- Verify prototype Variant Code is displayed using the `V##` format.

#### Document Validation

- Verify uploaded documents can be opened.
- Verify uploaded documents can be downloaded.
