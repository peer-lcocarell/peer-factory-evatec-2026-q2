# Test Case Title Format

```text
R10.2 Update for CR 286067 - <Requirement Name>: <Test Description> (<Test Type>)
```

## Naming Convention

- **R10.2 Update for CR 286067** = Release and Change Request reference
- **<Requirement Name>** = Requirement being validated
- **<Test Description>** = Specific functionality under test
- **<Test Type>** = Happy Path, Negative, Validation, Audit, Regression, Boundary, Error Handling, etc.
