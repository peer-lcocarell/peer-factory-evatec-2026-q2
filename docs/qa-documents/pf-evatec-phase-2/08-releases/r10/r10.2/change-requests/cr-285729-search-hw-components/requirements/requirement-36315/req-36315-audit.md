6.1.1REQUIREMENT: Tool Overview
https://ops1.peergroup.com/DefaultCollection/PFEvatec/_testPlans/define?planId=291616&suiteId=292238
The tool overview page will allow the user to see tools configured in PF and be the hub for performing tool activities. By default only active tools will be shown but the user can choose to view all tools (including removed).

Tools will be grouped by their ECL location and will show the name, tool type, owner, connection type, substrate size, position, production work order and equipment #, as well as live status and E10 stateinformation if applicable. The tool type, connection type and status can be filtered by multiple values.

The Details and History buttons are available if there is a selected tool and the user has the ToolDetails_View security right.

The Hardware and Search Hardware buttons are available if the selected tool has a logbook configured and the user has the HardwareConfiguration_View security right.

The Logbook button is available if there is a selected tool and the user has the Logbook_View security right.

Users must have the ToolOverview_View security right to access this screen.
