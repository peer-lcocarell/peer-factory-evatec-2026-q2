# CR 285729 – Search HW Components

## High-Level Summary

This CR introduces a new **Hardware Search** feature that helps users quickly locate where hardware components are installed within the factory configuration.

Users can search using:

- Part Number
- Description
- Serial Number
- Installation date range
- Docked or undocked module scope

Search results display the associated **Tool** and **Module** where the hardware is installed and provide navigation links that allow users to open those locations directly.

## Business Value

- Simplifies locating hardware across tools and modules.
- Reduces time spent manually reviewing hardware configurations and logbook records.
- Supports both current and historical hardware installation tracking.
- Improves troubleshooting and equipment management.

## Key Areas Impacted

- Hardware Search dialog
- Tools page
- Modules page
- Tool Overview
- Hardware installation history
- Logbook data

## QA Focus

- Search accuracy across all supported filters.
- Historical date-range searches.
- Current versus historical installation results.
- Tool and Module navigation links.
- Correct handling of module and submodule locations.

## Risk Level

**Medium**

The feature primarily introduces new search and reporting functionality but relies on historical hardware installation records and module hierarchy data. Incorrect data relationships could result in inaccurate search results or location reporting.

# Requirement Details

## Requirement 36315 – Tool Overview

### Impact of the Change

Tool Overview now supports navigation from Hardware Search results.

Users can:

- View search results associated with a Tool.
- Open Tool records directly from search results.
- Access hardware location information more efficiently.

### QA Considerations

Verify:

- Tool links open the correct Tool Overview page.
- Tool information displayed in search results is accurate.
- Historical search results reference the correct tool assignment.

---

## Requirement 226372 – Modules

### Impact of the Change

Module-related information is exposed through the Hardware Search feature.

Users can:

- View module locations associated with installed hardware.
- Open module records directly from search results.
- Search across docked and undocked modules.

### QA Considerations

Verify:

- Module links open the correct module.
- Submodule locations are reported correctly.
- Docked and undocked module filtering behaves as expected.
- Historical installation records reference the correct module location.

---

## Requirement 287451 – Search for Installed Hardware Components

### New Functionality

A new Hardware Search feature has been introduced.

Supported search criteria include:

- Part Number
- Description
- Serial Number
- Installation date range
- Docked modules
- Undocked modules

### Search Results

Results display:

- Hardware Component information
- Tool location
- Module location
- Navigation links to Tool and Module records

### Historical Tracking Support

The search feature supports:

- Current installations
- Historical installations
- Date-range filtering

### QA Considerations

Verify:

- Correct results are returned for all supported search fields.
- Multiple search criteria can be combined.
- Historical searches return expected results.
- Empty result sets are handled correctly.
- Navigation links open the correct records.
- Hardware installed in submodules is reported accurately.
