# Test Case Title
284111 Self-Service Box - Boxes List Required Columns and Self-Service Visibility (Happy Path)

## Requirement
- CR: 284111
- Ticket: 34682
- Requirement ID: 36297
- Requirement: Boxes list displays required columns, including Self-Service, and values match detail view.

## Preconditions

- User is signed in with `Boxes_View` permission.
- Box data includes at least one `IsSelfServe = true` and one `IsSelfServe = false` record.
- User is on the `Boxes` list page.

| Step | Action | Expected Result |
|------|----------|----------------|
| 1 | Open `Boxes` from main navigation. | List loads without errors and records are displayed. |
| 2 | Verify grid headers. | Required columns are visible, including `Self-Service`. |
| 3 | Locate a known self-service record. | `Self-Service` value displays `Yes` (or environment equivalent). |
| 4 | Locate a known non-self-service record. | `Self-Service` value displays `No` (or environment equivalent). |
| 5 | Open the self-service record from list. | Detail view opens and shows `Is Self-Serve = Yes`; values match list row. |
| 6 | Return and open non-self-service record. | Detail view opens and shows `Is Self-Serve = No`; values match list row. |
| 7 | Refresh the list page. | Required columns remain visible and `Self-Service` values remain consistent. |

## Expected Outcome

Boxes list consistently shows required columns, including `Self-Service`, and displayed values match box details.
